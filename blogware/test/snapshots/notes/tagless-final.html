Tagless-final encoding

✑2026-04-26

Tagless-final encoding started out as a solution to the Expression problem. It represents value types as module/trait functions and operations as module/trait instances.

The Shape example:

module type Shape = sig
  type t
  val rectangle : float -> float -> t
  val circle : float -> t
end

module Area = struct
  type t = float

  let rectangle w h = w *. h
  let circle r = Float.pi *. r *. r
end

module Perimeter = struct
  type t = float

  let rectangle w h = 2 *. (w +. h)
  let circle r = 2. *. Float.pi *. r
end

Adding new operations is straightforward, and adding the Triangle type doesn’t require modifying existing code:

module type ShapeExt = sig
  include Shape
  val triangle : float -> float -> t
end

module AreaExt = struct
  include Area
  let triangle base height = 0.5 *. base *. height
end
